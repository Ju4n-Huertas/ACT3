# Proyecto Web Modular con Flask

Aplicacion web desarrollada con Flask que implementa un diseño modular utilizando plantillas HTML con bloques de extension e inclusiones.

## Caracteristicas

- Estructura modular con plantilla base y bloques reutilizables
- Tres paginas principales: Inicio, Servicios y Contacto
- Componentes incluidos: Header, Navbar y Footer
- Estilos CSS personalizados con diseño responsive
- Navegacion dinamica con url_for de Flask

## Instalacion

Opcion A. Clonar el repositorio:
```bash
git clone https://github.com/Ju4n-Huertas/ACT3.git
cd ACT3
```
Opcion B. Descarga directa
  1. Descargar el proyecto como ZIP desde GitHub (boton "Code" -> "Download ZIP")
  2. Descomprimir el archivo
  3. Abrir la carpeta en VS Code
